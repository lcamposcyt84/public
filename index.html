API REST

Create your own marketplace with our API Rest. The Amacrux API Rest is a service provider for third parties at the Back-end level. dApps can connect to the Amacrux API to set content UX (User Experience) actions such as: importing, minting, listing and trading NFTs in a decentralized environment. The Rest API was built using CRUD operations: Create, Read, Modify and Delete; established under the Node.js architecture.

What is Amacrux

Improve on Amacrux

Amacrux will use smart contracts deployed on the BNB Chain network to automate functions characteristic of an NFT marketplace such as asset creation and import, auctions, item transactions, etc.

Amacrux is a universal open-source marketplace supported by developers, creators, and collectors. It facilitates trustless peer-to-peer trading of digital assets through an interface that provides dynamic tools for easy exploration, preparation, and launch of NFTs with real-world utility.

Secure Multi-Chain

Amacrux aspires to allocate multiple blockchain networks with the goal of creating a globalized and interoperable NFT market, enabling P2P asset trading using the most robust EVM-compatible chains in the crypto space.

Open Source

Amacrux code is a complete open source. Anyone can contribute to the codebase, run a node, propose updates for frontend and backend elements or build services on the platform. Currently, the source code is publicly available in GitLab repositories under the link: https://gitlab.com/amacrux
